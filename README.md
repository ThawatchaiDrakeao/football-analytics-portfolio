# ⚽️ Football Forward Efficiency Analysis (Goals vs xG)

**English** | [ภาษาไทย](#-บทสรุปภาษาไทย-thai-version)

## 📌 Project Overview
This project is a data analytics portfolio piece focusing on the finishing efficiency of top European football forwards. By comparing actual **Goals** scored against **Expected Goals (xG)**, the analysis identifies which players overperform their underlying metrics and demonstrate the highest clinical efficiency. 

## 🛠️ Tech Stack & Tools
* **Language:** Python 3
* **Libraries:** Pandas
* **Environment:** Jupyter Notebook / JupyterLab, VS Code
* **Core Concepts:** Data Cleaning, Feature Engineering (Calculating Finishing Efficiency), Exploratory Data Analysis (EDA)

## 📊 Executive Summary
Based on the statistical analysis comparing actual goal output against Expected Goals (xG) metrics, the key findings are:
1. **Top Finishing Efficiency:** Son Heung-min demonstrates the highest clinical efficiency among the analyzed players, overperforming his expected metrics by +2.9 goals.
2. **Mohamed Salah (Liverpool):** Consistently maintains elite-level performance, converting chances above the statistical baseline (+1.5 goals over xG), which underscores his high-caliber finishing capability.
3. **Erling Haaland:** Despite leading the scoring charts with 25 goals, he exhibits a slight underperformance relative to his xG (-1.2). This indicates a massive volume of high-value chances generated by his team's tactical system.
